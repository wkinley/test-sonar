使用方法
1、修改项目文件，commit
2、在sonarcloud.io查看检查结果
https://sonarcloud.io/organizations/wkinley/quality_profiles

若没有结果，在检查流水线在Actions里面查看
[https://sonarcloud.io/organizations/wkinley/quality_profiles](https://github.com/wkinley/test-sonar/actions)

2、在github配置集成sonar
